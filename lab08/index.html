Form Elements

Forms are a powerful tool to gather information from users on web pages. Here are five form elements (or input elements) that can be used to accept valuable data from people who are using your website.

Text Box

The text input element is used to collect names, phone numbers, titles, addresses, and other types of alphanumeric text.

Submit Button

The submit input element is used to submit the form itself. It is usually placed at the end of the form, and it's used when the user is done entering their data.

Check Box

The checkbox input element is used when multiple potential choices out of an available set of predetermined items should be selected.

Reset Button

The reset input element is used to reset the values of the form to its initial values. It does not submit the form.
